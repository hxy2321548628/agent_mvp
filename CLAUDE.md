# Implementing a Minimum Viable Agent from Scratch

This is a project to implement a minimum viable Agent (ReAct) that can use web search, weather query, and calculator tools.

## Constraints

### 1. Do not use existing agent frameworks (langgraph/openhands/openclaw). Build from scratch independently. You may draw on LangGraph's design patterns.

### 2. When writing code, place key parameters/variables at the top of the file; do not hardcode them inside functions.

### 3. Use the singular form for all file and folder names; do not use plurals.

### 4. When a final reply poses a question to the user, ask it through the question tool with selectable options — do not pose it as plain prose expecting the user to type a free-form answer.

### 5. The @src/ directory contains code related to the agent

### 6. @cli/ Client, using the agent in @src

### 7. Use mermaid to draw flowcharts

### 8. Reply to me in Chinese
